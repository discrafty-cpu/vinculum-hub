# Slide Builder — Vinculum Math Solutions

A browser-based presentation builder for creating auto-playing kiosk slideshows.

**Live at:** https://discrafty-cpu.github.io/slide-builder/

## Features
- Drag & drop PDF slides or images
- Embed live web apps (Google Apps Script, etc.) with individual timers
- Build a custom sequence with per-item durations
- Full-screen kiosk mode with progress bar and dot navigation
- Keyboard shortcuts (← → Space F Esc)
- No installation required — runs entirely in the browser

## Deploy Updates
```bash
cd ~/Web\ Apps/slide-builder
./deploy.sh
```
