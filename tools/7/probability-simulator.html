Key Concepts

Sample Space: All possible outcomes
Event: Specific outcome(s)
Probability: # favorable / # total
Experimental: From actual trials
Theoretical: Mathematically expected

How to Use

In Explore, click buttons to run 10 trials. See how experimental probability changes with each trial. Practice calculates probabilities from given scenarios.

Vocabulary

Sample Space: Set of all possible outcomes
Favorable Outcomes: Outcomes that satisfy event
Probability: Ratio of favorable to total
Independent: One event doesn't affect another
Compound: Two or more events
