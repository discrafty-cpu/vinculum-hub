Vocabulary

Data Information collected and organized, like measurements or counts.
Line Plot A graph showing data on a number line using X marks.
Bar Graph A graph using bars to show amounts and compare data.
Picture Graph A graph using pictures or symbols to show data.
Scale The numbers on a graph that show what the marks represent.
Key Information showing what each symbol or picture means.
