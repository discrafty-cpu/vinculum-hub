What is a Function?

A function is a relationship where each input has exactly one output. Think of it as a machine: you put something in, and get something out.

Is it a Function?

A relation is a function if each input has exactly ONE output. Use the vertical line test on a graph: if a vertical line crosses the graph more than once, it's not a function.
