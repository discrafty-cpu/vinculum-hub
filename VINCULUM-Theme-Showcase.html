How It Works — One Line Per App
Every app reads from localStorage.vinculum_theme. Adding theme support to any app = 1 line of JS + CSS variables already defined above.
// Add to EVERY app's <script> — one line: const theme = localStorage.getItem('vinculum_theme') || 'dark'; document.documentElement.setAttribute('data-theme', theme); // The CSS variables above go in a shared <style> block // or a vinculum-themes.css file imported by every app. // Hub saves the choice: function setTheme(t) { localStorage.setItem('vinculum_theme', t); document.documentElement.setAttribute('data-theme', t); }
