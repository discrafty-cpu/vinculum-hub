How to create your Google Form + Grading Spreadsheet:
  1. Go to script.google.com and click New Project
  2. Delete the default code and paste the copied script
  3. Click Run (play button). Authorize when prompted.
  4. Two items appear in your Google Drive:
    • Quiz Form — auto-grading, requires student email + name + period
    • Responses Spreadsheet — live grading dashboard with answer key, DOK analysis, class stats
  5. Share the form link with students. Results auto-populate the spreadsheet.
