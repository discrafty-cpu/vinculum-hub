PART 2: MINNESOTA 2022 ALIGNMENT
================================================================================

Minnesota 2022 Standards (effective 2027-28) reorganize K-12 math into:
  • Patterns & Relationships (P) — Number, Algebra combined
  • Spatial Reasoning (S) — Geometry, Measurement expanded
  • Data & Probability (D) — Statistics, Probability, Data integrated

KEY MN2022 CHANGES NOT YET ADDRESSED:

1. EARLY PROBABILITY (K-2)
   Status: ❌ Missing entirely
   MN2022: K.D.5, 1.D.5-6, 2.D experimental probability
   Tools Needed: "Fair Games" (K-1), "Probability Predictions" (2)

2. EARLY PROPORTIONAL THINKING (3-4)
   Status: ⚠️ Partial (mentioned in tools, not explicit)
   MN2022: 3-4 emphasize "growth patterns" and multiplicative relationships
   Tools Needed: "Growth Pattern Explorer" (3-4)

3. STATISTICAL PROCESS (K-8)
   Status: ⚠️ Partial (data displays present, process implicit)
   MN2022: Emphasis on full cycle (Question → Data → Analysis → Conclusion)
   Tools Needed: "Statistical Inquiry Planner" (5-8)

4. CULTURAL RESPONSIVENESS & AUTHENTIC CONTEXTS
   Status: ❌ Tools appear context-neutral
   MN2022: Requires "culturally sustaining pedagogy"
   Enhancement Needed: Add regional/culturally relevant contexts

5. SPATIAL REASONING LANGUAGE
   Status: ⚠️ Partial (tools use CCSS language, not "spatial reasoning")
   MN2022: Emphasizes "spatial thinking," "relationships," "location"
   Revision Needed: Relabel/reframe geometry tools to MN2022 terminology

PART 5: CRA PROGRESSION ANALYSIS
================================================================================

Concrete-Representational-Abstract progression assessment across K-12:

GRADE K-2: STRONG CONCRETE-REPRESENTATIONAL
  ✓ Excellent manipulatives (blocks, counters, physical shapes)
  ✓ Clear visual representational modes (diagrams, arrays, ten frames)
  ⚠️ Abstract modes underdeveloped (Play modes lack depth)
  Recommendation: Expand Play modes; clarify prerequisite chains

GRADE 3-5: TRANSITIONING TO REPRESENTATIONAL-ABSTRACT
  ✓ Concrete still present (division fair share, fractions)
  ✓ Strong representational (bar models, arrays, number lines)
  ⚠️ Abstract progressively dominant; missing some scaffolds
  Recommendation: Maintain concrete options for struggle learners

GRADE 6-8: ABSTRACT DOMINANT
  ❌ Concrete modes absent
  ⚠️ Representational modes adequate but uneven
  ❌ Abrupt R → A transitions in some tools
  Recommendation: Add representational bridges (balance models, visual proportions)

HIGH SCHOOL: ABSTRACT WITH MINIMAL REPRESENTATION
  ⚠️ Representational modes present (graphs, diagrams)
  ❌ No "multi-representation viewers" (simultaneous table/graph/equation)
  ❌ Proofs lack scaffolding/templates
  Recommendation: Enhance representational depth; add representation connections

MAJOR ISSUE: PREREQUISITE CHAINS UNDOCUMENTED
  Problem: No explicit documentation of which tool leads to which
  Example: After "Counting Objects Lab" (K), students don't know whether to
           take "Number Bonds," "Add & Subtract," or other tools
  Recommendation: Add prerequisite metadata; document learning pathways
