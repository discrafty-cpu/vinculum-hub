Indigenous Context Engine

This page tests the drop-in panel. Look at the green circle button in the bottom-right corner — click it to open the search panel.

How it works: The panel loads indigenous-data.json (84 curated entries) and matches your query using weighted scoring. No AI generates content — every story, problem, and cultural reference is pre-written from research.

Integration: One Line of Code

To add this panel to any page in your Hub or Digester, add this line before the closing </body> tag:

<script src="indigenous-context-panel.js"></script>

That's it. The panel appears automatically. The indigenous-data.json file must be in the same folder.

API-Only Mode (No Panel UI)

If you want to use the engine programmatically without showing the panel:

<script src="indigenous-context-panel.js" data-no-ui="true"></script>
<script>
  IndigenousContext.query({
    grade: "4",
    domain: "geometry",
    topic: "symmetry reflection"
  }).then(results => {
    console.log(results.primary.theme);
    // "Beadwork Design"
    console.log(results.primary.sample_problem);
    // "Design a beadwork pattern on a coordinate grid..."
  });
</script>
