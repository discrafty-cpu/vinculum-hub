VINCULUM Icon System
Replacing emojis with custom SVG icons
Emojis look different on every OS, can't match your theme colors, and feel "cheesy" at product-level. Below are 4 icon styles — each shown for the same 8 tools so you can compare directly. All are pure SVG (inline, no dependencies, theme-color aware).
Style A
Minimal Stroke
Thin 2px strokes, open shapes, no fills. Think Feather Icons or Lucide. Clean and lightweight — the icon doesn't compete with the math content. Works great at small sizes. Every icon uses currentColor so it automatically matches the theme text.
✓ Lightest visual weight — keeps focus on the math. ✓ Scales perfectly from 16px to 48px. ✓ Professional, modern feel.
✗ Can feel thin/weak at very small sizes. ✗ Less personality than filled styles.

Style B
Gradient Fill
Solid shapes filled with the theme gradient. Bold and eye-catching — the icons feel like part of the brand identity. Uses url(#vinculumGrad) SVG gradient definition. Similar to how Notion or Linear use filled accent icons.
✓ Strongest brand presence. ✓ Instantly recognizable. ✓ Works as app icons/favicons too.
✗ Heavier visual weight — may compete with content in some layouts. ✗ Gradient depends on SVG defs.

Style C
Duotone Glow
Two-layer icons: a soft, glowing background shape + a crisp foreground detail in the accent color. The glow layer uses the theme's --glow color. Feels premium and dimensional without being heavy. This is what Stripe and Vercel use.
✓ Best balance of weight and clarity. ✓ The glow layer adds depth. ✓ Very readable at all sizes.
✗ Slightly more complex SVG. ✗ Two colors per icon need to harmonize with every theme.

Style D
App Badge
Each icon sits inside a rounded square "badge" with the theme gradient background and a white/light foreground symbol. Think iOS app icons or Material Design filled icons. The badge shape gives every icon a consistent footprint so they align perfectly in grids.
✓ Most consistent grid alignment. ✓ Strongest visual "app" feel. ✓ Works perfectly as favicons and touch icons.
✗ Heaviest visual weight. ✗ The badge can feel rigid/corporate.
